## Expressões vs statements

- Expressões são pedaços de código que produzem um valor.
- Statements são pedaços de código que executam uma ação.

## Complemento

- Variávels podem ter vários tipos de dados
- Adicionar um a uma variável `x = x + 1`
- `+=`, `-=`
- `++`, `--`

## If statements

- Expressões booleanas
- if basico
  - {} vs sem {}
- `==` vs `===`
- outros operadores de comparação (>, <, >=, <=, !=, !==)
  - caso especial de ordenação de strings
- truthy e falsy values
- else e else if
- Operador ternário
